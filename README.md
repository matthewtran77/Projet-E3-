# Project E3 Conversion – Hydro-Québec

Project E3 is a large conversion initiative to migrate electrical single-line diagrams and related assets from AutoCAD to the E3 series.  
The project supports planning, operation, and maintenance activities at Hydro-Québec by standardizing electrical design tools and data.

---

## Purpose

This repository uses Git for version control to support development and validation work across:
The goal is to keep scripts, notes, and validation tools synchronized, stable, and easy to track over time.

---

## Scope of the Project

This repository may include:
- validation scripts
- automation tools
- configuration files
- documentation and notes related to E3 conversion
- testing or analysis utilities

Production data and confidential information are excluded.

---
### One repository = one project
This repository is dedicated to the E3 conversion project only.  
No unrelated experiments or personal code are included.

### Main branch stays stable
- `main` always contains working and verified code
- Never commit experimental or broken code directly to `main`
- New work is developed on feature branches
